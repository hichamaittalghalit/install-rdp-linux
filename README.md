
# Install RDP On Linux

<br>

### step 1:

<br>

    apt install git -y

<br>

### step 2:

<br>

    git clone https://github.com/hichamaittalghalit/install-rdp-linux/

<br>

### step 3:

<br>

    cd install-rdp-linux && sudo chmod 777 install.sh

<br>

### step 4: 

$1=login
$2=password

<br>

    sudo sh ./install.sh $1 $2
    
    ex: sudo sh ./install.sh your_login your_password
