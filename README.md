
# Install RDP On Linux

<br>

### step 1:

<br>

    apt install git -y

<br>

### step 2:

<br>

    git clone https://github.com/hichamaittalghalit/install-rdp-linux/

<br>

### step 3:

<br>

    cd install-rdp-linux && sudo chmod 777 install.sh

<br>

### step 4: 


<br>

    sudo sh ./install.sh
